The app and the data

This is simple and useful app that allows for daily monitoring the Costarican Coln exchange rate market. The available information is cumbersome to update and review on a daily basis. The source of the information is the Central Bank of Costa Rica

I order to streamline the updating of the data, all the data prior to end-of-year 2018 was dowloaded and kept in the server.

User information

The data is updated automatically for the convenience of the user and has a nice modern financial look to it. It includes a candlestick pricing chart and has 4 parameters that can me defined:

  • Initial date: The data can be displayed since December 2007. Normally these analyses are better for periods no longer than one year.
  • Weighted Average: allows for the plotting of the daily weighed average that is the actual reference rate.
  • Volume: Displays the bar chart with the daily volume in US$ millions.
  • Number of transactions: Displays the bar chart with the daily transactions.
